Software Produced from Our Research

MulVAL: A logic-based enterprise network security analyzer

The MulVAL project aims at providing automated tools to aid in security administration of enterprise networks. It builds upon a logic-based attack graph generator, leverages existing standard security information databases, and conducts comprehensive and quantitative analysis to help system administrators to harden the security of an enterprise network in a cost-effective manner.

SnIPS: Snort Intrusion Analysis using Proof Strengthening

This project focuses on the problem of how to automatically identify high-confident attack traces from large amounts of system monitoring data, through a simple logic with the capability of reasoning about uncertainty which is inherent in real-time security analysis.

ANCOR: Automated eNterprise network COmpileR

ANCOR is a cloud-based IT system generation and management framework. It uses a specification language called ARML (ANCOR Requirement Modeling Language) to define the high-level structure of an IT system, and compiles that high-level abstraction into concrete systems leveraging state-of-the-art cloud and configuration management technologies. A major advantage of ANCOR is that it keeps track of the dependencies among the deployed instances in the cloud and allows for easy adaptation of the IT system, opening doors for new security techniques such as moving-target defense.

Amandroid: A static analysis framework for Android apps

Amandroid is a precise and general inter-component data flow analysis framework for security vetting of Android apps. It builds an Inter-component Data Flow Graph (IDFG) which captures data flow across component boundaries, e.g., through intent passing. It also builds a data dependence graph (DDG) of the app from the IDFG. One can use this generic data-flow framework to perform various types of security analysis for Android apps, by writing a small plugin that captures the security properties of interest. Amandroid analysis is sound in that it creates an over-approximation of an app's behavior, under clearly defined and reasonable assumptions on the Android runtime environment.
