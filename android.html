Android Security Research

We conduct research to apply static analysis on Dalvik byte code of both Android applications and libraries, for the purpose of identifying potential malicious behaviors or program vulnerabilities. Our goal is two-fold: 1) the analysis needs to be sound, where the analysis result is an over approximation of an app's behavior with well specified assumptions on the app's runtime environments; 2) the analysis precision can be configured flexibly based on the available computing resources. Both goals are important to yield pracitcal Android app vetting tools that can be used by app market operators or companies to prevent potentially dangerous apps from getting onto users' devices.
